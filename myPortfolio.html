About Me

Saya bernama Muhammad Alfin Khaerudin. Saya lahir di Jakarta pada tanggal 01 Oktober 2003. Saya merupakan seseorang yang tertarik akan tentang dunia web developper dan editing. Dalam dunia web developper, beberapa bahasa pemrograman sudah saya kuasai dan dapat diimplementasikan selama saya berada di perkuliahan. Sedangkan dalam dunia editing saya sudah beberapa kali membuat project untuk seseorang. Berbagai macam perlombaan sudah saya lewati. Membuat suatu design dalam dunia editing adalah hal yang membuat saya senang. Saya dapat mengimplementasikan semua imajinasi saya dalam dunia editing. Jika dilihat secara permasalahan, kasus dalam dunia web developper juga sama halnya dengan dunia editing. Atas hal tersebutlah saya mencoba untuk menekuni dunia web developper dan dunia editing

Dalam melaksanakan pengerjaan project yang saya buat, saya merupakan tipe orang yang mengerjakan pekerjaan yang tidak ditunda-tunda. Saya mencoba utuk menanamkan dalam jati diri saya bahwa setiap menit dalam suatu waktu itu sangatlah berharga. Saya juga merupakan tiper orang yang ingin bekerja keras dan selalu berlandaskan kebaikan dalam kehidupan.
